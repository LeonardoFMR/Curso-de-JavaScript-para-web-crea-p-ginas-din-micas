En este curso se trabajó en la eraboración de funciones y automatización de las mismas, así como métodos de las funciones y un primer vistazo a la manipulación de elementos mediante el DOM.

Anotaciones:
¿El DOM es parte de HTML o de JavaScript? De hecho, de ninguno — él es generado por el browser. Al cargar la página, el navegador crea el documento, la interfaz, y Javascript usa el DOM para conectarse al HTML.
Para comunicarse entre ellos, es necesario insertar la etiqueta script en el archivo HTML, y como buena práctica, debe ser antes del cierre de la etiqueta body para que los scripts se carguen después del código base.
El DOM es el índice y como se muestra toda la página, siendo window el primer elemento, padre del document quien tiene como hijo el Root Elemet (HTML).
